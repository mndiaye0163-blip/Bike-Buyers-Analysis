#  Bike Buyers Analysis — Portfolio Project

##  Overview
This project analyses **1,026 customer records** to understand which demographic and lifestyle factors influence bike purchases.
The goal is to identify patterns across **age, income, region, commute distance, and occupation** to support targeted marketing and customer segmentation.

##  Dataset
The dataset contains **1,026 rows** and includes variables such as:

- Marital Status
- Gender
- Income
- Education
- Occupation
- Home Ownership
- Cars Owned
- Commute Distance
- Region
- Age
- Purchased Bike (Yes/No)

Example from the dataset:
> “Who buys bikes? Insights across Age, Income, Region, Commute & Occupation • n = 1,026 customers”
> “Total Buyers: 495 (48.2%)”


##  Data Cleaning
Steps performed:

- Removed duplicates
- Standardised income groups (Low, Mid, High, Very High)
- Created **Age Group** and **Income Group** columns
- Normalised commute distance categories
- Ensured consistent formatting for categorical fields

Example from cleaned data:
> “Age Group: 40–50, Income Group: Mid ($30–60k)”


## Tools Used
- **Excel** — initial cleaning
- **Power BI** — modelling & dashboard
- **DAX** — calculated measures & KPIs
- **Data Visualisation** — charts, slicers, insights

##  Key Insights
Based on the dashboard and dataset:

- **Buy rate is 48.2%** (495 out of 1,026 customers).
- Ages **30–50** show the highest purchase likelihood.
- **Mid‑income ($30–60k)** and **High‑income ($60–100k)** groups buy more bikes.
- The **Pacific region** has noticeably higher purchase volume.
- Shorter commute distances (0–1 miles) correlate with higher purchase rates.
- **Skilled Manual** and **Professional** occupations show strong buyer representation.

##  Dashboard Summary
The Power BI dashboard includes:

- Buy Rate by Age Group
- Buy Rate by Income Group
- Purchases by Region
- Purchases by Commute Distance
- Purchase Volume by Occupation
- Overall KPIs (Total Customers, Buyers, Avg Income)

##  How to Use

2. Explore interactive filters for age, region, income, and commute distance.
3. Review insights to understand buyer behaviour patterns.
